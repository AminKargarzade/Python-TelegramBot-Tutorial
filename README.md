# 🤖 Python Telegram Bot Tutorial

A practical repository for learning how to build Telegram Bots using Python and the **pyTelegramBotAPI (telebot)** library.

This project contains simple examples that demonstrate how to interact with the Telegram Bot API and serves as a starting point for developing more advanced Telegram bots.

---

## ✨ Features

- Telegram Bot development with Python
- Uses the `pyTelegramBotAPI` library
- Environment variable support for secure bot token management
- Logging enabled for easier debugging
- Clean and beginner-friendly project structure
- Test files for experimenting with different media types

---

## 🛠 Tech Stack

- Python 3
- pyTelegramBotAPI
- Telegram Bot API

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/AminKargarzade/Python-TelegramBot-Tutorial.git

cd Python-TelegramBot-Tutorial
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ⚙ Environment Variables

Create the following environment variables before running the bot.

| Variable | Description |
|----------|-------------|
| API_TOKEN | Your Telegram Bot Token |
| CHANNEL_ID | Telegram Channel ID (optional depending on the example) |

Example:

```bash
API_TOKEN=xxxxxxxxxxxxxxxx
CHANNEL_ID=-1001234567890
```
